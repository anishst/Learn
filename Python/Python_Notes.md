## Logging
https://docs.python.org/3/library/logging.html

Basic Usage
```
FORMAT = '%(asctime)s:%(levelname)s:%(message)s:%(module)s:%(filename)s:%(filename)s'
logging.basicConfig(filename='test.log',level=logging.INFO,format=FORMAT)
```
