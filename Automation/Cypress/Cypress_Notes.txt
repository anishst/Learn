Install
https://docs.cypress.io/guides/getting-started/installing-cypress.html

API Commands: https://docs.cypress.io/api/api/table-of-contents.html

1. Go to project folder using command window
2. Run: npm init

Follow prompts; example below

            package name: (cypress_test)
            version: (1.0.0)
            description: test cypress
            entry point: (index.js)
            test command:
            git repository:
            keywords:
            author: Anish
            license: (ISC)
            About to write to C:\Users\ats\Documents\cypress_test\package.json:

            {
            "name": "cypress_test",
            "version": "1.0.0",
            "description": "test cypress",
            "main": "index.js",
            "scripts": {
                "test": "echo \"Error: no test specified\" && exit 1"
            },
            "author": "Anish",
            "license": "ISC"
            }


3. Run: npm install cypress --save-dev
4. Cypress gets installed under: C:\Users\ats\AppData\Local\Cypress\Cache\3.1.5\Cypress
5. To launch cypress use: ./node_modules/.bin/cypress open

Sample script: test_google.specc.js

    describe('My first cypress test', () => {
        it('Navigate to google site', () =>{
            cy.visit('http://www.google.com')
        })
    })

To remove cypress:  npm uninstall cypress

npm run cy:run -- --spec "cypress/integration/examples/google_test.spec.js"
